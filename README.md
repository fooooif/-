# SELD-Visualization-Program
DCASE2020을 활용한 졸업작품(시각화 프로그램)

### 프로젝트 배경
 - 딥러닝 기술을 활용한 소리분석 연구의 발전
 - 일상 생활속 소리분석 활용 가능
 - 4차 산업 분야에서의 필요성 증가
 - DCASE2020 연구 자료를 활용
